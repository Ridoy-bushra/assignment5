    - 1️⃣ What is the difference between var, let, and const?
    ans:1. Difference between var, let, and const:

var, let, and const are used to declare variables in JavaScript.
var is the old way of declaring variables. It is function-scoped and can be updated and re-declared, which can sometimes create problems.
let is a modern way. It is block-scoped, meaning it works only inside a block (like inside {}). It can be updated but cannot be re-declared in the same scope.
const is also block-scoped, but it is used for values that should not change. Once a value is assigned, it cannot be updated or re-declared. - 2️⃣ What is the spread operator (...)?
